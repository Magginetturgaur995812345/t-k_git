cd ..#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	a:cd = fer inn í möppu
	b:cd ~ = fer tilbaka á heimasvæði
	c:ls = sýnir þér innihald möppu
	d:pwd = segir þér hvaða möppu þú ert í
	e:mkdir = býr til möppu
2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	a: git clone = klónar innihald í nýja skrá
	b: git log = sýnir historíið 
	c: git status = Birtir leiðir sem hafa mismun milli vísitölu skráar og núverandi yfir geranda
	d: git diff = Sýnir breytingar milli vinnandi tré og vísitölu tré
	e: git checkout = Uppfærir skrár í vinnu trénu til að passa við útgáfu í vísitölu eða tiltekins tré
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff (Sýnir breytingar milli vinnandi tré og vísitölu tré)
	b) git diff --staged (synir breytingar á nú þegar breyttum hlutum)
	c) git diff commit1 commit2 (sýnir mismun á sumum eða öllum möppum)
4. 	Hvað er útgáfustýring (e.version control)? 
	stjórnandi á skjala breytingum
5.	Hverjir eru helstu kostir við að nota GIT?
	þú getur unnið í verkefninu sjálfur, samsetning er auðveld, verkefnið er sveigjanlegt
6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	eftir hverja spurningu, til að sjá breytingar eftir hverja.
7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	vinnu svæði, sviðsetningar svæði og geymsla.
