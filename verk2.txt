1. Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.
Random accses memory og það er gagnasöfnun sem hækkar hraða á vélinni þinni.
2. Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er.
Read only memory og það guit
3. Hvað er fyrirbyggjandi viðhald?
Fara yfir tölvuna t.d þrífa ryk og fara yfir skjöl
4. Fyrir hvað stendur skammstöfunin ESD, hvað er það og hvaða leiðir höfum við til að
verjast því þegar unnið er með tölvubúnað?
Electric static discharge  og maður á að losa sig við stöðurafmagn til að verjast því
